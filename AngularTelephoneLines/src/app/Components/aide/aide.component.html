Aide d'utilisation

FAQ - Lignes Téléphonique

Question : Comment puis-je ajouter une nouvelle ligne téléphonique ?

Réponse : Cliquez sur le bouton "AJOUTER" en haut du tableau. Un formulaire s'ouvrira pour vous permettre de saisir les informations de la nouvelle ligne.

Question : Comment filtrer les lignes téléphoniques ?

Réponse : Utilisez le champ "Filter" pour entrer un mot-clé. Le tableau sera mis à jour pour afficher uniquement les lignes correspondant à votre recherche.

Question : Comment puis-je modifier une ligne existante ?

Réponse : Sélectionnez la ligne que vous souhaitez modifier, puis cliquez sur l'icône "edit" (crayon) qui apparaît à côté de la ligne.

Question : Comment puis-je supprimer une ligne ?

Réponse : Sélectionnez la ligne que vous souhaitez supprimer, puis cliquez sur l'icône "delete" (corbeille) qui apparaît à côté de la ligne.

Question : Que signifie "Aucune donnée pour le filtre" ?

Réponse : Cela signifie qu'aucune ligne ne correspond à votre critère de recherche dans le champ "Filter".

Question : Comment puis-je zoomer dans le tableau ?

Réponse : Utilisez les boutons de zoom "+" et "-" en bas du tableau pour ajuster le niveau de zoom.

Question : Que faire si je rencontre une erreur lors de l'ajout, de la modification ou de la suppression d'une ligne ?

Réponse : Si une erreur se produit, un message s'affichera en bas de l'écran. Suivez les instructions fournies dans le message ou contactez le support technique.

FAQ - Types de Lignes Téléphoniques

Question : Comment puis-je voir tous les types de lignes téléphoniques disponibles ?

Réponse : Vous pouvez voir tous les types de lignes téléphoniques dans le conteneur de types. Chaque bouton représente un type de ligne différent.

Question : Comment passer au type de ligne suivant ou précédent ?

Réponse : Utilisez les boutons de navigation (flèches gauche et droite) situés aux deux extrémités du conteneur de types.

Question : Comment ajouter une nouvelle ligne téléphonique ?

Réponse : Cliquez sur le bouton "AJOUTER" situé en haut à droite de la table. Une fenêtre s'ouvrira pour vous permettre d'ajouter une nouvelle ligne.

Question : Comment filtrer les lignes téléphoniques dans la table ?

Réponse : Utilisez le champ "Filtre" situé en haut à gauche de la table pour rechercher des lignes spécifiques.

Question : Comment modifier ou supprimer une ligne téléphonique ?

Réponse : Cliquez sur une ligne dans la table pour la sélectionner. Des boutons d'action "Modifier" et "Supprimer" apparaîtront en bas de la table.

Question : Comment zoomer dans la table ?

Réponse : Utilisez les boutons de zoom situés en bas à droite de la table pour ajuster le niveau de zoom.

Question : Que faire si je vois un message d'erreur ou un écran de chargement ?

Réponse : Si un message d'erreur s'affiche, veuillez contacter le support technique. Si l'écran de chargement persiste, veuillez attendre un moment ou actualiser la page.

Question : Comment puis-je changer le nombre de lignes affichées par page dans la table ?

Réponse : Utilisez le menu déroulant en bas à gauche de la table pour sélectionner le nombre de lignes à afficher par page.

Question : Comment naviguer entre les différentes pages de la table ?

Réponse : Utilisez les boutons de pagination situés en bas à droite de la table pour naviguer entre les pages.

FAQ - Composant Corbeille

Question : Comment puis-je rechercher une ligne spécifique dans la corbeille ?

Réponse : Utilisez le champ "Recherche..." situé en haut de la corbeille pour rechercher des lignes spécifiques.

Question : Comment filtrer les lignes par date ?

Réponse : Utilisez le champ "Choisissez une date..." situé en haut de la corbeille pour filtrer les lignes par date.

Question : Comment restaurer une ligne supprimée ?

Réponse : Cliquez sur l'icône "restore_from_trash" (restaurer depuis la corbeille) à côté de la ligne que vous souhaitez restaurer.

Question : Comment supprimer définitivement une ligne de la corbeille ?

Réponse : Cliquez sur l'icône "delete" (supprimer) à côté de la ligne que vous souhaitez supprimer définitivement.

Question : Comment changer le nombre d'éléments affichés par page ?

Réponse : Utilisez le menu déroulant "Éléments par page :" en bas de la corbeille pour choisir le nombre d'éléments à afficher.

Question : Comment naviguer entre les différentes pages de la corbeille ?

Réponse : Utilisez les boutons de pagination situés en bas de la corbeille pour naviguer entre les différentes pages.

Question : Comment restaurer ou supprimer tous les éléments de la corbeille ?

Réponse : Utilisez les boutons "Restaurer toute la corbeille" ou "Supprimer toute la corbeille" situés en bas de la corbeille pour effectuer ces actions sur tous les éléments.

Question : Que faire si un message d'erreur ou un écran de chargement s'affiche ?

Réponse : Si un message d'erreur s'affiche, il sera indiqué dans une zone spécifique de la corbeille. Si l'écran de chargement persiste, veuillez attendre un moment ou actualiser la page.

FAQ - Importation de la Base de Données

Question : Comment puis-je importer un fichier Excel ?

Réponse : Cliquez sur le bouton "Importer un fichier excel" et sélectionnez le fichier Excel que vous souhaitez importer.

Question : Quels formats de fichier sont acceptés ?

Réponse : Le fichier Excel doit être au format .xlsx ou .xls.

Question : Que faire si j'obtiens un message d'erreur lors de l'importation ?

Réponse : Vérifiez que le fichier respecte tous les critères mentionnés dans la section d'instructions. Si le problème persiste, contactez le support technique.

Question : Comment naviguer entre les différents types de lignes ?

Réponse : Utilisez les boutons de navigation "Premier type", "Type précédent", "Type suivant" et "Dernier type" pour naviguer entre les différents types de lignes.

Question : Que signifie "Type de ligne non sélectionné" ?

Réponse : Cela signifie que vous n'avez pas sélectionné de type de ligne pour l'importation. Vous devez en sélectionner un pour procéder.

Question : Comment annuler l'importation ?

Réponse : Cliquez sur le bouton "Annuler" pour réinitialiser le processus d'importation.

Question : Comment vérifier le résultat de l'importation ?

Réponse : Une fois l'importation terminée, un résumé apparaîtra, indiquant le nombre d'enregistrements sauvegardés, vides et non sauvegardés.

FAQ - Profil Utilisateur

Question : Comment puis-je modifier mon nom d'utilisateur ?

Réponse : Cliquez sur le bouton "Modifier les informations ?" et modifiez le champ correspondant au nom d'utilisateur.

Question : Comment puis-je changer mon mot de passe ?

Réponse : Cliquez sur le bouton "Changer le mot de passe ?" et suivez les instructions.

Question : Quelle est la longueur minimale du mot de passe ?

Réponse : La longueur minimale du mot de passe est de 6 caractères.

Question : Que se passe-t-il si les deux mots de passe ne correspondent pas ?

Réponse : Vous obtiendrez un message d'erreur indiquant que les mots de passe ne correspondent pas.

Question : Comment puis-je annuler les modifications ?

Réponse : Cliquez sur le bouton "Annuler" qui se trouve à côté du bouton "Valider" dans les formulaires de modification.

Question : Que se passe-t-il si je reçois un message d'erreur lors de la modification des informations ?

Réponse : Assurez-vous que tous les champs sont correctement remplis et respectent les critères de validation. Si le problème persiste, contactez le support technique.

FAQ - Rapprochement

Question : Comment fonctionne le rapprochement automatique ?

Réponse : Le rapprochement automatique compare les données entre les différentes sources et identifie les correspondances.

Question : Que se passe-t-il si les données ne peuvent pas être rapprochées ?

Réponse : Un message d'erreur s'affiche, vous indiquant la raison pour laquelle le rapprochement a échoué.

Question : Que faire si le rapprochement prend trop de temps ?

Réponse : Si le processus de rapprochement est lent, assurez-vous que les serveurs sont opérationnels et que la charge réseau est minimale. Si le problème persiste, contactez le support technique.

Question : Comment puis-je exporter les résultats du rapprochement ?

Réponse : Utilisez le bouton "Exporter les données vers un fichier Excel" pour télécharger un fichier contenant les données rapprochées.
